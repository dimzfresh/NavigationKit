🚀 NavigationKit

A powerful SwiftUI navigation framework with coordinator pattern

Overview

NavigationKit is a modern SwiftUI navigation framework that provides a clean, coordinator-based architecture for managing complex navigation flows in iOS applications. Built specifically for iOS 16+ with full SwiftUI support.

View on GitHub Documentation

Key Features

🎯 Coordinator Pattern

Clean separation of navigation logic from view presentation using the coordinator pattern.

🔒 Type-Safe Navigation

Compile-time safety for all navigation operations with strong typing.

📱 Multiple Presentation Styles

Support for push, sheet, full-screen, and custom presentation styles.

🔗 Deep Link Support

Built-in deep link handling with easy integration.

⚡ SwiftUI Native

Designed specifically for SwiftUI with iOS 16+ features.

🛡️ Thread-Safe

MainActor compliance ensures thread-safe navigation operations.

// Add to your Package.swift

Basic Usage

@MainActor
final class MainCoordinator: BaseCoordinator {
    @Published var childCoordinator: (any Coordinator)?
    @Published var navigationRouters: [NavigationRouter] = []
    
    var rootView: some View {
        NavigationStack(path: rootNavigationRouter.pathBinding) {
            HomeView()
                .navigationDestination(for: NavigableView.self) { navigableView in
                    navigableView.view
                }
        }
        .onAppear {
            setupNavigationRouter()
        }
    }
}
